# Estatus de la Salud Mental en Adultos del Municipio más Acaudalado de América Latina.

Este repositorio contiene un reporte técnico detallado sobre el desarrollo de modelos avanzados de clasificación — **Random Forest Classifier**, **Boosting**, **Support Vector Machine (SVM)** y **Red Neuronal construida desde cero** — para clasificar el estado de salud mental de adultos en edad económicamente activa del municipio de **San Pedro Garza García, Nuevo León**. El estudio retoma el dataset y las variables trabajadas a lo largo del segundo parcial, aplicando la técnica de balanceo **SMOTE** e incorporando nuevos modelos de mayor complejidad para comparar su poder predictivo.

El proyecto utiliza técnicas de métodos de ensamble, máquinas de soporte vectorial y redes neuronales, basándose en datos recopilados del **Portal de Datos Abiertos del Gobierno del Estado de Nuevo León**.

## Estructura del repositorio.

- **Actividad 2.3: Modelos de Ensamble, SVM y Red Neuronal**: Implementación y comparación de Random Forest, Boosting, SVM y Red Neuronal, en escenario desbalanceado y balanceado con SMOTE.
  - `saludMental_Avanzado.ipynb`: Notebook principal con el código, análisis y modelos.
  - `saludMental_Avanzado.html`: Reporte en formato HTML para visualización web.
