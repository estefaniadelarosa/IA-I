# Análisis de la Obesidad en Países Latinoamericanos.

Este repositorio contiene un estudio estadístico sobre los factores que influyen en el IMC en poblaciones de **Colombia, Perú y México**.

El proyecto utiliza técnicas de aprendizaje estadístico y automático para identificar variables determinantes en la salud pública de la región, basándose en datos recolectados por investigadores de la Universidad de la Costa, Colombia.

## Estructura del Repositorio.

* **[Actividad 1.1: Aprendizaje Estadístico-Automático](./A1.1_Aprendizaje_Estadistico/)**: Análisis exploratorio, limpieza de datos y pruebas de correlación sobre el conjunto de datos de obesidad.
    * `Obesidad.ipynb`: Notebook principal con el código y reporte técnico.
    * `Obesidad.html`: Reporte en formato HTML.
    * `A1.1_Obesidad.csv`: Base de datos utilizada.
