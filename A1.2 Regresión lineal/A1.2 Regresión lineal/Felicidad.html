1.1 Introduccin.

Para el presente estudio de la felicidad segn distintas caractersticas, se emplear primero un modelo de regresin lineal simple con la variable de GDP, para ms adelante emplear ahora un modelo de regresin lineal mltiple con varias variables en juego - este se va a realizar con tres variables aparte del GDP, sabiendo que esas tres son las que presentan una mayor correlacin con la felicidad.

En cada uno de los modelos, se separan los datos para entrenar el modelo en datos de entrenamiento y de prueba, esto para dar pie a reportar los R2 respectivos para conocer si el modelo es confiable y los errores para conocer si el modelo es preciso tanto en la visualizacin de la inferencia como del modelo como tal. Tambin se aade como un extra la prediccin de cada modelo segn cierto valor de las variables utilizadas.

La recopilacin de los datos viene por el medio de Blackboard para la regresin lineal simple, para la eventual regresin lineal mltiple, estos vienen del World Happiness Report; ambas fuentes reportan datos del 2022.

1.2 Objetivo.

El objetivo de este estudio es conocer cules son los factores que afectan al nivel de felicidad de los individuos en ciertos pases, esto puede ser significativo para las entidades tanto pblicas como privadas de dichos pases con el propsito de conocer a su poblacin y conocer qu mejoras se necesitan implementar para subir los niveles de satisfaccin en estas variables y por ende, tener mayores niveles de felicidad para evitar problemas sociales, econmicos, migratorios, etc.

# Aspecto de inferencia. 
y_pred = modelRLS.predict(x_test) # Preddiccin.

residuals = y_test - y_pred # Errores.

# RSS (Residual Sum of Squares).
RSS = np.sum(residuals**2)

# TSS (Total Sum of Squares).
TSS = np.sum((y_test - np.mean(y_test))**2)

# RSE (Residual Standard Error).
n = len(y_test)
p = 1 # Solo 1 variable.
RSE = np.sqrt(RSS / (n - p - 1))

print("RSS =", RSS)
print("TSS =", TSS)
print("RSE =", RSE)
