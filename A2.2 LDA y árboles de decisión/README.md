# Estatus de la Salud Mental en Adultos del Municipio más Acaudalado de América Latina.

Este repositorio contiene un reporte técnico detallado sobre el desarrollo de modelos de **Análisis Discriminante Lineal (LDA)** y **Árbol de Decisión** para clasificar el estado de salud mental de adultos en edad económicamente activa del municipio de **San Pedro Garza García, Nuevo León**. El estudio retoma el dataset y la clasificación trabajados previamente en la Regresión Logística, incorporando la técnica de balanceo **SMOTE** identificada como la más efectiva, y extiende el análisis con dos nuevos modelos de clasificación.

El proyecto utiliza técnicas avanzadas de aprendizaje automático, incluyendo reducción de dimensionalidad, poda de árboles y validación de modelos, basándose en datos recopilados del **Portal de Datos Abiertos del Gobierno del Estado de Nuevo León**.

## Estructura del repositorio.

- **Actividad 2.2: LDA y Árbol de Decisión**: Implementación de LDA para reducción de dimensionalidad y separación de clases, más Árbol de Decisión original y podado con técnica SMOTE.
  - `LDA-Tree_SPGG.ipynb`: Notebook principal con el código, análisis y modelos LDA y Árbol de Decisión.
  - `saludMental_Avanzado.html`: Reporte en formato HTML para visualización web.
