# Predicción de Calificaciones Escolares.

Este repositorio contiene un reporte técnico detallado sobre el desarrollo de un modelo de **Regresión Lineal Múltiple** para predecir las calificaciones finales de estudiantes. El estudio analiza cómo diversos factores demográficos y sociales influyen en el desempeño académico basándose en datos de dos instituciones educativas.

El proyecto utiliza técnicas de aprendizaje estadístico y automático para identificar variables determinantes en el éxito educativo, basándose en datos recopilados del **UCI Machine Learning Repository**.

## Estructura del repositorio.

* **[Actividad 1.3: Modelos de Regresión Múltiple](./A1.3_Modelos_Regresion_Calificaciones/)**: Implementación de limpieza de datos, análisis de multicolinealidad (VIF) y modelo de regresión para predecir el desempeño estudiantil.
    * `calificacionesFinales.ipynb`: Notebook principal con el código, análisis y modelo de regresión múltiple.
    * `calificacionesFinales.html`: Reporte en formato HTML para visualización web.
    * `student-mat.csv`: Base de datos utilizada para el entrenamiento del modelo.
