# Estatus de la Salud Mental en adultos del municipio más acaudalado de América Latina.

Este repositorio contiene un reporte técnico detallado sobre el desarrollo de un modelo de **Regresión Logística** para clasificar el estado de salud mental en adultos del municipio de San Pedro Garza García, Nuevo León. El estudio analiza la probabilidad de que un individuo pertenezca a grupos de trastornos específicos o requiera servicios de salud preventivos, basándose en factores demográficos y clínicos.

El proyecto utiliza técnicas de aprendizaje estadístico para realizar una clasificación binaria, identificando si una persona padece una enfermedad que requiere tratamiento profesional o si requiere terapia de apoyo. Los datos provienen del portal de datos abiertos del Gobierno del Estado de Nuevo León (periodo octubre 2024 - agosto 2025).

## Estructura del repositorio.
Actividad: Modelo de Regresión Logística: Implementación de limpieza de datos enfocada en el rango de edad de 18 a 65 años, análisis de desbalance de clases y validación del modelo de clasificación.

regresionLogistica_SM-SPGG.ipynb: Notebook principal con el código, análisis exploratorio, tratamiento de desbalance y modelo de regresión logística.

regresionLogistica_SM-SPGG.html: Reporte en formato HTML para visualización web.
