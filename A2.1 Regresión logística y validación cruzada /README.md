# Estatus de la Salud Mental en Adultos del Municipio más Acaudalado de América Latina.

Este repositorio contiene un reporte técnico detallado sobre el desarrollo de un modelo de **Regresión Logística** para clasificar el estado de salud mental de adultos en edad económicamente activa del municipio de **San Pedro Garza García, Nuevo León**. El estudio analiza cómo variables demográficas y clínicas permiten predecir si un individuo padece un trastorno mental que requiere atención profesional especializada, o si sus condiciones solo implican factores de riesgo manejables.

El proyecto utiliza técnicas de aprendizaje estadístico y automático, incluyendo selección de características, manejo de desbalance de clases y validación cruzada, basándose en datos recopilados del **Portal de Datos Abiertos del Gobierno del Estado de Nuevo León**.

## Estructura del repositorio.

- **Actividad 2.1: Regresión Logística**: Implementación de limpieza de datos, análisis de multicolinealidad (VIF), manejo del desbalance de clases y modelo de clasificación binaria.
  - `regresionLogistica_SM-SPGG.ipynb`: Notebook principal con el código, análisis y modelo de regresión logística.
  - `saludMental_Avanzado.html`: Reporte en formato HTML para visualización web.
